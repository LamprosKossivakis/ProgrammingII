ΟΔΗΓΙΕΣ

(1) Στο branch main υπάρχει το αρχείο COVID_19_DB.sql το οποίο περιέχει όλεσ τις απαραίτητεσ γραμμές
κώδικα για την εγκατάσταση της βάσης τοπικά μαζί με όλα τα απαραίτητα εργαλεία για την λειτουργεία της
execute αυτού του αρχείου σε SQL Server

(2) για την λειτουργεία του προγράμματοσ απαιτείται η αλλαγή των username και password στην κλάση ConnectionPool 
και όπου αλλου χρησημοποιείται ο jdbc driver, σε αυτά που έχει ορίσει ο χρήστης στον sql server του

(3) το jar περιέχει όλα τα dependencies που απαιτούνται(jdbc) 

(4) τα στατιστικά έχουν δημιουργηθεί με την βοήθεια του προγράμματοs tableau πάνω σε μία δοκιμαστική βάση
και στο branch main συμπεριλαβάνονται screenshots απο το περιβάλλον της ανάπτυξης των διαγραμμάτων

(5) Ο πηγαίος κώδικας βρίσκεται στο branch master
